## 기능 목록
### 입력
- [ ] 서로 다른 3자리 수를 받는 입력
- [ ] 재시작 1 / 종료 2 입력
- [ ] 예외처리 IllegalArgumentException
  - [ ] 숫자가 서로 다르지 않는 경우
  - [ ] 숫자가 아닌 경우
  - [ ] 3자리가 아닌 경우
  - [ ] 게임이 끝나고, 1, 2가 아닌 경우
### 출력
- [ ] 볼, 스트라이크 개수 (예: 1볼 1스트라이크)
- [ ] 하나도 없는 경우 낫싱
- [ ] 게임 시작 문구 출력
- [ ] 정답 메세지, 게임 종료 출력
- [ ] 종료 후 재시작, 종료 선택 메세지 출력


### 컴퓨터 숫자 선택
- [ ] 서로 다른 랜덤 숫자 3개 선택 
### 결과 판정
- [ ] 스트라이크 개수 구하기 (자리가 똑같고, 숫자도 똑같아야 됨)
- [ ] 볼 개수 구하기 (자리는 틀리고, 숫자가 똑같아야 됨)
- [ ] 낫싱 판정 (볼, 스트라이크 개수가 없는 경우)
